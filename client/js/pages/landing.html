Bookmarker

This is a web app that allows users to manage their bookmarks online. Bookmarks can be organized into pages and categories. Users can add, edit, move and delete their bookmarks.

This web app is built with the MEAN stack with AngularJS on the front-end and Node.js, Expree and MongoDB on the backend.

Author: LabsterX

Github Link: https://github.com/labsterx/bookmarker
